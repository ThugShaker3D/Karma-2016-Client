# Karma's Game Client.
Aesthetiful's Finobe is an attempt to revive the classic finobe.com experience.

## Requirements for building this game client
To build the source code, you will need MSVC 2019.

### Libraries
 - Boost 1.73.0
 - cpp-netlib 0.11.0
 - DSBaseClasses
 - QT 5.15
 - DirectX SDK
 - SDL 2.0.14
 - sgCore
 - VMProtectWin 3.5.1
 - w3c-libwww 5.4.0
 - curl 7.43.0
 - zlib 1.2.8
